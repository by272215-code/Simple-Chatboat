# 🤖 Basic Chatbot

## Introduction

This project is a simple rule-based chatbot built using Python. It was developed as part of the **CodeAlpha Python Programming Internship**.

The chatbot interacts with users through a command-line interface and responds to predefined messages. It can answer basic greetings, introduce itself, and end the conversation when the user types "bye".

This project helped me understand how chatbots work at a basic level and how Python can be used to create interactive applications using loops, functions, and conditional statements.



## Project Overview

The chatbot takes input from the user and responds based on predefined rules. It continuously runs until the user decides to end the conversation.

Although simple, this project demonstrates the core logic behind conversational applications and provides a foundation for building more advanced AI-powered chatbots in the future.



## Features

* Interactive command-line chatbot
* Responds to common greetings
* Answers basic questions
* Runs continuously until the user exits
* User-friendly conversation flow
* Beginner-friendly Python implementation


## Technologies Used

* Python 3
* Functions
* Loops
* Conditional Statements (if-elif-else)
* User Input and Output



## How It Works

1. The chatbot starts and displays a welcome message.
2. The user enters a message.
3. The chatbot checks the input against predefined responses.
4. If a matching response is found, the chatbot replies accordingly.
5. If the input is not recognized, the chatbot displays a default message.
6. The conversation continues until the user types **"bye"**.
